Como Usar o Mapa

📍 Navegação no Mapa

Zoom: Use a roda do mouse ou os botões + / - para aproximar/afastar

Mover: Clique e arraste para mover o mapa

Marcadores: Clique nos pinos vermelhos para ver informações detalhadas

🔍 Buscando Hospitais

1. Use a caixa de busca no canto superior esquerdo do mapa

2. Digite o nome do hospital, cidade ou estado

3. Pressione Enter para encontrar

📱 Versão Mobile

• Toque e arraste com o dedo para navegar

• Use dois dedos para fazer zoom

• Toque nos marcadores para ver detalhes

Como Buscar Hospitais

🏥 Métodos de Busca

  • Por Nome: Digite o nome completo ou parcial do hospital
  • Por Cidade: Ex: "New York", "Los Angeles", "Chicago"
  • Por Estado: Ex: "California", "Texas", "Florida"
  • Por CEP: Digite o código postal da região

🎯 Dicas de Busca

• Use palavras-chave específicas

• Tente variações do nome (ex: "NYU" ou "New York University")

• Combine cidade + especialidade

• Use a busca por proximidade para encontrar hospitais próximos

Filtros Disponíveis

📊 Tipos de Filtros

  • Localização: Estados, cidades, regiões
  • Especialidades: Internal Medicine, Surgery, Pediatrics, etc.
  • Tipo de Instituição: Academic, Community, Private
  • Tamanho do Hospital: Pequeno, médio, grande

⚙️ Como Aplicar Filtros

1. Clique no ícone de filtro no mapa

2. Selecione os critérios desejados

3. Clique em "Aplicar" para ver os resultados

4. Use "Limpar Filtros" para resetar

💡 Dica Pro

Combine múltiplos filtros para resultados mais precisos!

Informações de Contato

📧 O que Você Encontrará

  • Email do Coordenador: Contato direto para aplicações
  • Telefone: Número para contato telefônico
  • Endereço Completo: Localização exata do hospital
  • Website Oficial: Link para mais informações
  • Requisitos: Pré-requisitos para aplicação

📝 Informações Importantes

• Sempre verifique os requisitos específicos

• Alguns programas têm deadlines específicos

• Confirme a disponibilidade antes de aplicar

• Mantenha documentos atualizados

⚠️ Atenção

As informações são atualizadas regularmente, mas sempre confirme diretamente com o hospital.

Dicas para Sucesso

🎯 Antes de Aplicar

  • Pesquise bem o hospital e programa
  • Prepare todos os documentos necessários
  • Tenha seu CV atualizado em inglês
  • Prepare uma carta de apresentação personalizada

📋 Documentos Essenciais

  • CV em inglês atualizado
  • Diploma médico traduzido
  • Cartas de recomendação
  • Certificado de proficiência em inglês
  • Seguro médico (malpractice insurance)

💌 Dicas de Comunicação

  • Seja profissional e cordial
  • Personalize cada email
  • Seja claro sobre suas intenções
  • Siga up após 2-3 semanas se não houver resposta

🏆 Aumentando suas Chances

  • Aplique para múltiplos programas
  • Tenha flexibilidade nas datas
  • Mostre experiência relevante
  • Demonstre motivação genuína

Sobre a Ward Academy

🎓 Nossa Missão

A Ward Academy é dedicada a apoiar médicos brasileiros em sua jornada para observerships nos Estados Unidos. Acreditamos no lema "De OLD para OLD" - compartilhando experiências e conhecimento entre colegas.

🌟 O que Oferecemos

  • Mapa interativo com hospitais verificados
  • Informações atualizadas de contatos
  • Dicas e orientações especializadas
  • Suporte na jornada USMLE
  • Comunidade de médicos brasileiros

📊 Nossos Números

• Mais de 500 hospitais mapeados

• Centenas de médicos já beneficiados

• Informações constantemente atualizadas

• Suporte 24/7 via redes sociais
